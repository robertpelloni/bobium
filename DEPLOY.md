# DEPLOY: bobium

To deploy and test bobium, the project must be compiled from source on a local heavy-compute machine.

## Build Prerequisites
- **Disk:** 150GB+ free SSD space.
- **RAM:** 32GB+ (64GB recommended).
- **Compute:** 8+ Core CPU.
- **Tools:** `depot_tools` in `$PATH`.

## Execution
Refer to `docs/BUILD_INSTRUCTIONS.md` for step-by-step fetch, patch, and compilation commands.
